Distance in the plane

This web page can be used in order to calculate the distance between two points: P(\(x_1,y_1\)) and Q(\(x_2,y_2\))

The distance between two points in the plane is computed by this formula:

$$ dist(P,Q) = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2} $$
